# 코딩 테스트 코드 구현 노하우.


- 조기반환
```python

det total_price(quantity, price):
  total = quantity * price
  if total > 100:
    return total * 0.9
  return total
  
print(total_price(4,50))

```

- 합성함수
```python
def add_three(x):
  return x + 3
  
def square(x):
  return x * x
 
composed_function = lambda x: square(add_three(x))
print(composed_function(3))

```
